# Monophone 📱✨

Monophone is a premium, distraction-free minimalist launcher built using Flutter and Node.js. It is designed to help you reclaim your focus, drastically reduce screen time, and streamline your daily mobile workflow by replacing addictive visual clutter with an intentional, text-based ecosystem.

## 🚀 Download & Installation (v1.0.1)

To get Monophone running on your device right now, download our optimized production binaries directly from our official release pipeline below:

* 🌟 **[Download Monophone (Optimized 64-bit APK)](https://github.com/HarshDikshit/monophone/releases/download/v1.0.1/app-arm64-v8a-release.apk)** — **Recommended** for almost all modern Android devices (including Samsung Galaxy, Pixel, and OnePlus handsets) for the best performance and battery efficiency.
* 📦 **[Download Monophone (Universal Fallback APK)](https://github.com/HarshDikshit/monophone/releases/download/v1.0.1/app-release.apk)** — Contains all CPU architectures in a single, slightly larger file size.

### 📲 How to Install:
1. Tap one of the download links above directly from your Android device.
2. Open the downloaded `.apk` file from your device's notification shade or File Manager.
3. If prompted by Android, allow your file manager or browser to *"Install apps from unknown sources"*.
4. Tap **Install**, open the application, and set Monophone as your default launcher!

## 🛠️ System Architecture & Tech Stack

* **Frontend:** Flutter & Dart (Ahead-of-Time compiled for maximum execution speed).
* **Backend:** Express.js & Node.js (Hosted securely on Render cloud infrastructure).
* **Database:** MongoDB Atlas (Production-grade global cluster shards).
